JavaScript WebAPI

JavaScript 是什么

运行在浏览器的编程语言

组成

  • ECMAScript：语言基础
  • WebAPI：DOM 操作文档 + BOM 操作浏览器

书写位置

  1. 内部 JS - HTML 文件里，用 script 标签包住，script 标签写在 </body> 上面
  2. 外部 JS - 用 <script src=""></script>
  3. 内联 JS - 写在标签内部

输入输出语法

输入

  • 页面弹出输入框：prompt("请输入")
  • 取到的是字符串类型，要取数字可以用 +prompt（隐式转换）

输出

  • document.write('内容') - 向 body 内输出内容，如果是标签会被解释为网页元素
  • alert('内容') - 弹窗
  • console.log('内容') - 控制台打印输出，程序员调试用

变量

  • 变量声明：let 变量名，可以用逗号隔开多个
  • 赋值：变量名=字面量
  • 本质：内存地址
  • 规则规范：和 C 差不多

常量

  • 用 const 声明的变量
  • 声明的时候必须要赋值
  • 值不会改变，不允许更改

数组

let arr = [value1, value2, value3]

  • 索引从 0 开始
  • 类型不一定要一样
  • 长度：数组名.length = 索引号 + 1

模板字符串：`...${变量}...` 内的东西，用 ${} 使用变量

数据类型

基本数据类型（存值）

  • 数字、字符串（单双引号）、布尔、未定义、空
  • 放到栈里

复杂数据类型（存地址）

  • Object、Array、Date
  • 放到堆里

运算符

基础运算：+, -, *, /, %

  • 不正常的计算会得到 NaN
  • NaN 的一切计算都是 NaN

赋值运算符：+=, -=, 等等（左边必须是容器）

一元运算符：+, -, ++, --（分前后置）

比较运算符：>, <, >=, <=, ==, ===（值和类型是否都一样）, !==

  • 结果是 boolean 类型 true/false
  • 字符串比较就比字典序
  • 有 NaN 比较全是 false
  • 小数最好别比较，会有精度问题

逻辑运算符：&&, ||, !

优先级

  1. （）
  2. 一元 包括 !
  3. 算数
  4. 关系
  5. 相等
  6. 逻辑（&& > ||）
  7. 赋值
  8. 逗号

流程控制语句

分支

if 分支

if() // 自动转 boolean
{
}
else
{
}
// 除了 0 空字符串都为真

三元运算符：条件 ? 表达式1 : 表达式2

switch：使用 break 退出，default 为默认情况

循环

while 循环 - 三要素：起始值、终止值、变化量

do-while 循环

for 循环

退出循环：break（中止），continue（继续下一次）

函数

声明语法

function 函数名() {
    函数体
}

调用语法

函数名()

函数传参

  • 形参可以有默认值
  • 没传默认是 undefined，计算结果会是 NaN
  • 可以用数组作为参数，给个默认值空数组

返回值

  • 使用 return 语句
  • 会结束函数，没写的默认返回 undefined

函数作用域

  • 全局变量 VS 局部变量
  • 形参是局部变量
  • 如果在函数内部的变量没有声明那么默认当成全局变量
  • 变量的访问原则：先找当前作用域，没有再找上一级作用域

匿名函数

  • 无法直接调用
  • 赋值给变量用：let fn = function() { }
  • 立即执行函数：(function() { })() 必须加分号

逻辑中断

用来给默认值

  • && 都真时返回后一个真值
  • || 不都为假时返回前一个真值

Boolean 转换

0、undefined、null、false、NaN 转 boolean 都是 false，其余为 true

对象

Object：一种数据类型，无序数据的集合

声明

  • let 对象名 = {}
  • let 对象名 = new Object()

组成

属性和方法

增删改查

  • 查：对象.属性 或 对象["属性"]
  • 改：对象.属性=新值
  • 删：delete 对象.属性

方法

对象中的函数叫方法，可以通过 对象.方法() 调用

遍历对象

for(let key in obj) {
    console.log(obj[key])
}

内置对象

随机数：Math.random()（0-1）
获取 0-10 的随机整数：Math.floor(Math.random()*(1+10))

DOM 和 BOM

JS 操纵浏览器和 HTML

分类

  • DOM：文档对象模型，操作网页内容
  • BOM：浏览器对象模型

CSS 选择器选中 DOM 对象

const box = document.querySelector('div')
const box = document.querySelector('.box')
const box = document.querySelector('#box')
  • 返回值：匹配的第一个元素，也就是对象
  • querySelectorAll()：匹配多个，返回伪数组

操作元素内容

  • innerText 属性：获取/设置文本内容
  • innerHTML 属性：获取/设置 HTML 内容，会解析标签

操作元素属性和样式

操作常用属性

对象.属性 = 值 有就是改，没有就是增

操作样式属性

  1. 通过 style 属性操作 CSS：对象.style.样式属性 = 值
    如果是 background-color 这种的话用小驼峰命名法 backgroundColor
  2. 操作类名操作 CSS：box.className = 'box'
  3. classList 操作类控制 CSS：
    • 元素.classList.add('类名') 追加
    • 元素.classList.remove('类名') 删除
    • 元素.classList.toggle('类名') 切换

操作表单元素属性

  • .value 获取值
  • .checked 获取单选框是否被选中（返回布尔）
  • .disabled 禁用按钮（true）

自定义属性

// 以 data- 开头的
<div data-id="1" data-spm="2"> </div>
one.dataset 可以获取自定义属性列表
one.dataset.id 获取要的东西

间歇函数

setInterval(函数名, 间隔时间)
// 不要加（）！要不然就 '函数()'
  • 返回值是个数字，表示自己是第几个定时器（id）
  • 关闭定时器：clearInterval(n)
  • 重新赋值就可以再开

事件监听

元素对象.addEventListener('事件类型', 要执行的函数)
// 或者
元素对象.on事件 = 函数

事件类型

click鼠标点击
mouseenter鼠标经过
mouseleave鼠标离开
focus获得焦点
blur失去焦点
keydown按下触发
keyup抬起触发
input输入事件

环境对象 this

代表当前函数运行环境

const btn = document.querySelector('button')
btn.addEventListener('click', function(){
    console.log(this)  // this 是指向 btn
})

回调函数

将函数 A 作为参数传给函数 B 时，函数 A 就是回调函数

事件捕获和冒泡

  • 事件捕获：从 DOM 根目录依次执行事件
  • 事件冒泡：从底部往回（默认）
  • addEventListener('click', function(){}, true) - true 为捕获，false 为冒泡

阻止冒泡

  • btn.onclick = null
  • btn.removeEventListener('click', fn)

mouseenter 和 mouseleave 没有冒泡（推荐使用）

事件委托

给父元素注册事件，触发子元素的时候会冒泡到父元素身上。用事件对象可以判断是哪个子元素触发的

ul.addEventListener('click', function(e){
    if(e.target.tagName == 'LI') {
        e.target.style.color = 'red'
    }
})

组织默认行为

比如表单没填完整不让提交：对象.preventDefault()

其他事件

页面加载事件

window.addEventListener('load', function(){
    // 得加载完了才能有按钮
})
// 或者
document.addEventListener('DOMContentLoaded', function(){
    
})

页面滚动事件

window.addEventListener('scroll', function(){
    
})
  • .scrollTop 和 .scrollLeft 可以知道滚了多少
  • 获取整个页面用 document.documentElement

页面尺寸事件

  • 获取宽高：clientWidth 和 clientHeight 不包括边框
  • offsetWidth 和 offsetHeight 包括边框
  • 获取位置：offsetTop 和 offsetLeft，以最近一级的有定位的祖先元素为基准

日期对象

const date = new Date()
const date = new Date('2022-05-01 08:30:00')
  • date.toLocaleString() 获取本地日期时间字符串
  • date.getTime() 获取时间戳

时间戳

1970 年至现在的毫秒数

  • date.getTime()
  • +new Date()
  • Date.now()

节点查找

父节点查找

.parentNode

子节点查找

.children 得到伪数组

兄弟节点

  • .previousElementSibling
  • .nextElementSibling

增加节点

const li = document.createElement('li')
const ul = document.querySelector('ul')
li.innerHTML = 'content'
ul.appendChild(li)  // 插到最后面
ul.insertBefore(li, ul.children[0])  // 插到指定位置

克隆节点

.cloneNode(布尔值) - true 连子代一起，false 只复制自己（默认）

删除节点

父节点.removeChild(子元素)

M 端事件

touchstart、touchmove、touchend

Window 对象

顶级对象，代表浏览器窗口

延时函数

只执行一次：setTimeout(回调函数, 时间)
清除：clearTimeout(延时函数)

Location

  • 跳转 URL：location.href=""
  • 刷新：location.reload()，加 true 为强制刷新

Navigate

查看系统型号

History

历史记录：back()、forward()、go(±1)

JS 执行机制

同步异步

  • 耗时的是异步，放在任务队列里（点击等事件，资源加载，定时器）
  • 其他是同步，在执行栈里

执行顺序：先执行同步任务 → 再按时间执行异步任务（放进执行栈里）

本地存储

localStorage

  • 关闭浏览器不消失
  • 只能存字符串
  • 存：localStorage.setItem('键','值')
  • 取：localStorage.getItem('键')
  • 删：localStorage.removeItem('键')

sessionStorage

关闭浏览器就消失

存数组和对象

// 转化为 JSON 字符串存进去
localStorage.setItem('键', JSON.stringify(obj))
// JSON 字符串转对象
JSON.parse(localStorage.getItem('obj'))

Array 方法

map 方法

映射，返回值是新的数组

arr.map(function(ele, index){
    return ele + '要加的字符串'
})

join 方法

连接成一个字符串，（）里用小括号可以加上分隔符

arr.join('')  // 空字符串是没有分隔符

filter 方法

筛选数组中符合要求的
